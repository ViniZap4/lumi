# Welcome to lumi

This is your first note! lumi is a local-first note-taking system.

## Features

- **Plain Markdown files** - Your notes are just files on disk
- **Vim-like navigation** - Use j/k/h/l to navigate
- **External editor** - Edit in your favorite editor
- **Realtime sync** - Changes sync across clients

## Getting Started

1. Press `j` and `k` to navigate
2. Press `e` to edit this note
3. Press `n` to create a new note
4. Press `?` for help

Happy note-taking!
